# Glimmer Project Documentation

This directory contains all design, architectural, and workflow documentation for the Glimmer project.

### [**Design Documentation**](./design/README.md)

This section contains the core architectural and workflow documents that define the Glimmer platform. It covers the "Sessions, Agents, and Buckets" model, how agents are implemented, and the detailed user flows for key creative tasks.

### [**Agent Interaction Sequence Diagram**](./agent_interaction_sequence.md)

This document contains a detailed Mermaid sequence diagram illustrating the end-to-end flow of an interaction between the CUI, the backend, and the specialized agents. It specifically diagrams the "character not found" scenario and its resolution.
